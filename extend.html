建設資金問題
光是在西雅圖的一家 Amazon Go 就燒掉了近100萬美金，如果按照 Amazon 希望擴展的目標數量來看，成本至少需要30億美金，因為大部分都是高科技產品，就需要大量資金的配置，所以無人商店地點的選擇就成了一大考題，目前無人商店的地點都設在擁擠的大城市中，主要是和便利商店爭奪客源。不過即使方便複製展店的模式，因成本的考量，大量快速展店的目標可得在稍等一段時間了。


內部技術問題

(1) 人數管制問題
 以西雅圖總部舉例，總店能承受的最大人數為92人，一旦超出這個限制，攝影鏡頭會無法追蹤及定位顧客，會導致商品、價格計算出現錯誤，所以時常管制出入的數量而導致大排長龍，大大影響了消費者的購物體驗。

(2) 商品拿取機制
 在現行的技術下，機器的運行機制認為「拿取者」即為「購買者」，機器只能判斷此商品被何人拿走，並無法判斷代拿或轉交的行為，所以在 Amazon Go 中，可別好心的替其他顧客拿取商品，否則這筆金額將會算在你的身上。

歧視敏感議題
Amazon Go 最大賣點莫過於不必排隊掏錢可以拿了就走，但不收現金的政策卻也涉及歧視，Amazon 遭質疑是歧視沒有信用卡或銀行帳戶的低收入戶、少數族群或移民人士，對不熟悉科技的年長者也不友善，為保障低所得消費者等權益，費城在今年 3 月初簽署新法，成為美國首個禁止無現金商店的城市，拒收現金的商家將面臨最高 2,000 美元罰鍰。同月，紐澤西也通過法案，禁止當地設立無現金商店，另外包括紐約市、舊金山、芝加哥和華府也都計劃跟進，使得 Amazon Go 的未來發展遭受極大阻礙。


建議改善之處

引進電子貨架標籤 ( ESL )
 在 Amazon Go 中，員工有一項工作是：更換商品標籤，那如果換成更方便的電子標籤呢？可以省下印製及紙材成本，也可以減少員工的作業時間，而電子標籤上可以放上星數、顧客評語、QR Code 產品履歷等等資訊，若是有促銷活動時也可用來推播商品優惠訊息，可說是百利而無一害。
